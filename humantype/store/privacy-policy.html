Chrome Extension · Privacy Policy

HumanType

Effective date: June 21, 2025  ·  Last updated: June 21, 2025

The short version HumanType runs entirely on your device. It collects no data, sends nothing to any server, and has no analytics, accounts, or telemetry. The text you paste into the extension never leaves your computer.

What HumanType does

HumanType is a Chrome extension that types your pasted content into any focused web text field with realistic human timing — variable speed, natural pauses, typos, and self-corrections. It uses Chrome's debugger API to dispatch keystroke events into the tab you select during the arm window.

Data collection

HumanType collects no user data of any kind. Specifically:

  • The text you paste into the editor is never stored beyond your current session and never transmitted anywhere.
  • No personally identifiable information is collected.
  • No browsing history, page content, or URLs are read or recorded.
  • No keystrokes are logged. The extension generates keystrokes — it does not record them.
  • No analytics, crash reports, or telemetry are sent.
  • There are no user accounts and no sign-in.

Local storage

The extension stores one preference in your browser's localStorage: your chosen UI theme (light or dark). This value never leaves your device.

Permissions

HumanType requests the following Chrome permissions, each used solely for its stated purpose:

Permission Why it is needed
activeTab Identifies which browser tab the user wants to type into. Used only to get the tab ID so the debugger can attach to it.
sidePanel Displays the extension's UI as a side panel that stays open while the user clicks their target text field. A popup would close on click, making the workflow impossible.
debugger Dispatches real keystroke events into the active tab via Chrome's Input.dispatchKeyEvent API. This is the only way to reliably type into modern editors like Google Docs. The debugger attaches only to the tab the user selects, is used solely to send keystrokes, and detaches the moment the session ends. No page content is read, no network traffic is observed.
content_scripts (<all_urls>) Injects a small script on all pages to listen for the Alt+P keyboard shortcut, which lets the user pause or resume typing without switching to the side panel. The script reads no page content and makes no network requests.

Third-party services

The extension's side panel loads the Inter font family from Google Fonts (fonts.googleapis.com). This is a CSS stylesheet, not executable code. No other external services, APIs, or third-party scripts are used. No data is shared with any third party.

Children's privacy

HumanType does not collect any data from any user, including children. It is not directed at children under 13.

Changes to this policy

If this policy changes, the updated version will be posted at this URL with a new effective date. Because HumanType collects no data, changes are unlikely.
